# Cytron AI Coding Pack — URC10 Sumo Robot Controller

Welcome to the **Cytron AI Coding Pack for URC10**!

This pack is designed to help you **continue learning after completing the basic URC10 Sumo Robot tutorial**. It gives you all the reference material, example code, and AI-ready prompts you need to modify, debug, expand, and truly understand your URC10 robot.

---

## Who is this for?

- Students and hobbyists who have **completed the official URC10 getting-started tutorial**.
- Anyone who wants to **customize their sumo robot** beyond the default sample code.
- Learners who want to use an **AI coding assistant** (like Antigravity) to help them write and understand Arduino code.

## Before You Start

> **Prerequisite:** Complete the official URC10 tutorial first.
> This pack assumes your board is already set up, drivers are installed, and you have successfully uploaded the sample code at least once.

---

## Hardware

- **Board:** Cytron URC10 Sumo Robot Controller
- **Microcontroller:** ATmega328P (Arduino Uno compatible)
- **Language:** Arduino C/C++
- **IDE:** Arduino IDE
- **Motor Library:** Cytron Motor Drivers Library
